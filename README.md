# Car Price Prediction using Machine Learning

This project predicts car prices using regression techniques like **Linear Regression**, **Random Forest**.  
Data preprocessing, feature engineering, and model comparison are performed using Python, Scikit-learn, and Pandas.

- Linear Regression  
- Random Forest Regressor
